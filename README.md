## **Документация**

### **Введение**
Это приложение EvaCalendar, предназначенное для отслеживания событий по неделям, включая задачи и финансовые расходы.

### **Структура Проекта**
Проект разделен на две основные части:
- `back/`: Бэкенд-приложение (API) на Node.js с использованием Express.js.
- `front/`: Фронтенд-приложение на React с использованием Vite и TypeScript.

### **Бэкенд (`back/`)**

**Технологии:**
- Node.js
- Express.js (веб-фреймворк)
- Knex.js (конструктор запросов SQL, используется для миграций)
- SQLite3 (база данных)
- Jest (тестирование)
- PM2 (продакшн-менеджер процессов)

**Установка:**
1. Перейдите в директорию `back/`: `cd back/`
2. Установите зависимости: `npm install`
3. Запустите миграции для создания таблиц базы данных: `npx knex migrate:latest`
4. Запустите сидеры для заполнения начальных данных (если есть): `npx knex seed:run`

**Запуск:**
- **Режим разработки:** `npm start` (или `node index.js`)
- **Режим продакшена (с PM2):** `pm2 start pm2.config.js`

**Архитектура:**
- **Контроллеры (`controllers/`):** Обрабатывают HTTP-запросы и вызывают соответствующие сервисы.
- **Сервисы (`services/`):** Содержат бизнес-логику и взаимодействуют с базой данных.
- **Миграции (`migrations/`):** Управляют схемой базы данных.
- **`db.js`:** Модуль для подключения к базе данных.
- **`index.js`:** Входная точка приложения, настройка маршрутов и middleware.

**API Эндпоинты (примеры):**
- `GET /weeks`: Получить информацию о неделях.
- `POST /weeks`: Создать новую неделю.
- `GET /tasks/:weekId/:dayOfWeek`: Получить задачи для определенного дня.
- `POST /tasks`: Создать новую задачу.
- `GET /expense-categories`: Получить список категорий расходов.
- `GET /summary/:weekId`: Получить сводку заработка и трат за неделю.

### **Фронтенд (`front/`)**

**Технологии:**
- React (UI-библиотека)
- Vite (инструмент сборки)
- TypeScript (язык программирования)
- CSS (стили)
- Jest / React Testing Library (тестирование)

**Установка:**
1. Перейдите в директорию `front/`: `cd front/`
2. Установите зависимости: `npm install`

**Запуск:**
- **Режим разработки:** `npm run dev`
- **Сборка для продакшена:** `npm run build` (создает папку `dist/`)

**Архитектура:**
- **Компоненты (`src/components/`):** Переиспользуемые UI-элементы (например, `WeekView`, `TaskItem`, `TaskForm`).
- **Сервисы (`src/services/api.ts`):** Взаимодействуют с бэкенд API.
- **`src/main.tsx`:** Точка входа для React-приложения.
- **`src/App.tsx`:** Корневой компонент приложения.

**Основные Компоненты:**
- `WeekView`: Отображает неделю, разделенную на две половины.
- `FirstHalfOfWeek` / `SecondHalfOfWeek`: Отображают дни недели с задачами.
- `DayColumn`: Колонка для определенного дня недели.
- `TaskItem`: Отображает отдельную задачу.
- `TaskForm`: Форма для создания/редактирования задач.
